## Background
Analysis of differentially expressed genes between breast cancer 
and normal tissue using public RNA-seq data from NCBI GEO.

## Dataset
- Source: NCBI GEO (GSE109169)
- Samples: Breast cancer vs normal tissue

## Methods
- Differential expression: DESeq2 (R)
- Visualization: ggplot2, pheatmap (R), matplotlib (Python)
